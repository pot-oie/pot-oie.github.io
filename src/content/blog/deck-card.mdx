> 在找扑克牌 icon 时才发现，英文语境中扑克牌不叫 “Poker” ，“Poker” 只是扑克牌的一种玩法
>
> 扑克牌：Playing Card
>
> 一副扑克牌：a deck of playing cards

尝试在“最近看过”板块中除了常规的 grid 视图外再添加 deck 视图：

先让 Gemini 生成了一次，初具雏形，但是效果很差

```css
// in /src/styles/global.css
.movie-deck-container {
    position: relative;
    width: 100%;
    height: 360px; 
    display: flex;
    justify-content: center;
    align-items: flex-end;
    padding-bottom: 20px;
}
.movie-deck {
    position: absolute;
    bottom: 0;
    list-style-type: none;
    padding: 0;
    margin: 0;
    display: flex;
    justify-content: center;
    align-items: flex-end;
    width: auto; 
    height: 300px;
}
.movie-deck-card {
    position: absolute;
    aspect-ratio: 2 / 3;
    width: 200px;
    height: 300px;
    border-radius: 8px;
    overflow: hidden;
    box-shadow: 0 5px 20px rgba(0, 0, 0, 0.4);
    cursor: pointer;
    transition: transform 0.4s cubic-bezier(0.25, 0.8, 0.25, 1), box-shadow 0.3s ease;
    transform-origin: bottom center;
    --base-angle: -15deg;
    --angle-step: 6deg;
    --rotation: calc(var(--base-angle) + var(--i) * var(--angle-step));
    --base-x-offset: -120px;
    --x-step: 60px;
    --x-translation: calc(var(--base-x-offset) + var(--i) * var(--x-step));
    --y-translation: 0px; 
    transform: 
        translateX(var(--x-translation)) 
        translateY(var(--y-translation)) 
        rotate(var(--rotation));
}
.movie-deck-card img {
    width: 100%;
    height: 100%;
    object-fit: cover;
}
.movie-deck-container:hover .movie-deck-card {
    --base-angle: -20deg;
    --angle-step: 10deg;
    --rotation: calc(var(--base-angle) + var(--i) * var(--angle-step));
    --base-x-offset: -180px;
    --x-step: 90px;
    --x-translation: calc(var(--base-x-offset) + var(--i) * var(--x-step));
    transform: 
        translateX(var(--x-translation)) 
        translateY(var(--y-translation)) 
        rotate(var(--rotation));
}
.movie-deck-container .movie-deck-card:hover {
    transform: 
        translateX(0)
        translateY(-60px)
        rotate(0deg)
        scale(1.15);
    z-index: 10;
    box-shadow: 0 10px 30px rgba(0, 0, 0, 0.6);
}
.movie-deck-card .card-content {
    position: absolute;
    inset: 0;
    padding: 1rem;
    display: flex;
    flex-direction: column;
    justify-content: flex-end;
    background: linear-gradient(to top, rgba(0,0,0,0.9) 0%, rgba(0,0,0,0.5) 50%, transparent 100%);
    opacity: 0;
    transition: opacity 0.3s ease;
    color: white;
    border-radius: 8px;
}
.movie-deck-card:hover .card-content {
    opacity: 1;
}
.movie-deck-card .card-title {
    font-size: 1.1rem;
    font-weight: 700;
    line-height: 1.2;
}
.movie-deck-card .card-rating {
    display: flex;
    align-items: center;
    font-size: 0.9rem;
}
```
