> 作为开发者，经常需要在博客文章中展示代码片段。
>
> 一个好的代码块样式不仅能提升阅读体验，还能让代码看起来更专业和美观。
>
> 我自己尝试实现了类似 MacOs 代码块的视觉效果，分享给大家。

在实现美化代码块的同时添加了行号&语言类型显示和复制代码功能：

需要注意的是，Astro v5 之后代码块行号记录不再能通过 config 显式开启。

原来只需 `numberedLines: true`
```css
// in /src/styles/global.css
/* 模拟 macOS 窗口栏 */
pre.astro-code::before {
  content: '';
  position: absolute;
  top: 0;
  left: 0;
  right: 0;
  height: 1.75rem;
  background-image: 
    radial-gradient(circle at 1.5rem 1.25rem, #ff5f56 0.375rem, transparent 0.375rem),
    radial-gradient(circle at 2.75rem 1.25rem, #ffbd2e 0.375rem, transparent 0.375rem),
    radial-gradient(circle at 4rem 1.25rem, #27c93f 0.375rem, transparent 0.375rem);
  background-repeat: no-repeat;
  z-index: 1;
}
/* 语言提示 */
pre.astro-code::after {
  content: attr(data-language); // 这个属性取决于你的具体框架
  position: absolute;
  top: 0.9rem;
  right: 3.5rem;
  padding: 0.125rem 0.5rem;
  border-radius: 0.25rem;
  font-size: 1.25rem;
  font-weight: 600;
  text-transform: uppercase;
  z-index: 1;
  display: inline-block;
}
/* 复制按钮'mingcute--copy-line' */
/* 我的常用 ICON 包，可替换 */
pre.astro-code .copy-btn {
  position: absolute;
  top: 0.4rem;
  right: 1rem;
  width: 2rem; 
  height: 2rem;
  padding: 0.125rem;
  border: none;
  border-radius: 0.25rem;
  background-color: transparent;
  cursor: pointer;
  z-index: 3;
  transition: background-color 0.2s;
  display: flex;
  align-items: center;
  justify-content: center;
  outline: none;
}
/* 复制按钮 hover */
pre.astro-code .copy-btn:hover {
  background-color: rgba(0, 0, 0, 0.5);
}
/* 复制成功提示'mingcute:check-fill' */
pre.astro-code .copy-success {
  position: absolute;
  top: 0.4rem;
  right: 1rem;
  width: 2rem;
  height: 2rem;
  padding: 0.125rem;
  border-radius: 0.25rem;
  z-index: 4;
  opacity: 0;
  transition: opacity 0.2s;
  display: flex;
  align-items: center;
  justify-content: center;
  pointer-events: none;
}
/* 复制成功时显示 */
pre.astro-code .copy-success.show {
  opacity: 1;
}
/* 行号 */
pre.astro-code > code {
  display: block;
  width: 100%;
  padding: 3.5rem 1.5rem 1.5rem 3rem;
  counter-reset: line;
  box-sizing: border-box;
}
/* Shiki 生成的每一行 */
/* Astro 默认 */
.astro-code .line {
  display: block;
  position: relative;
  max-width: 100%;
  box-sizing: border-box;
}
/* 行号生成 */
.astro-code .line::before {
  content: counter(line);
  counter-increment: line;
  position: absolute;
  top: 0;
  left: -3.3rem;
  width: 2.5rem;
  text-align: right;
  color: inherit;
  opacity: 0.7;
  user-select: none;
  pointer-events: none;
}
```
